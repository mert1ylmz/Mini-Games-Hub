# 🎮 Mini Games Hub

Modern ve şık bir arayüze sahip, eğlenceli ve klasik mini oyunlar koleksiyonu.

## 🌟 Özellikler

-   **Modern UI/UX**: Glassmorphism etkileri, yumuşak geçişler ve Inter fontu ile premium bir görünüm.
-   **Kütüphane Bağımlılığı Yok**: Saf (Vanilla) JavaScript, HTML ve CSS ile geliştirildi.
-   **Responsive Tasarım**: Mobil cihazlarda ve masaüstünde kusursuz deneyim.
-   **Akıllı Oyunlar**: Tic-Tac-Toe için Minimax algoritması ile güçlendirilmiş yenilmez AI.

## 🕹️ İçerikteki Oyunlar

| Oyun | Emoji | Açıklama |
| :--- | :---: | :--- |
| **Snake** | 🐍 | Klasik yılan oyunu! Yemleri ye, büyü ama kendine çarpma. |
| **Tic-Tac-Toe** | ❌ | AI'ya karşı XOX oyna. Minimax algoritması ile yenilmez rakip! |
| **Memory Match** | 🃏 | Kartları çevir ve eşlerini bul. Hafızanı test et! |
| **Flappy Bird** | 🐦 | Boruların arasından geç! Basit ama bağımlılık yapıcı. |
| **2048** | 🔢 | Sayılarını kaydır ve birleştir. Hedef: 2048 taşına ulaşmak! |

## 🛠️ Kullanılan Teknolojiler

-   **HTML5**: Semantik yapı.
-   **CSS3**: Modern değişkenler (`:root`), Flexbox, CSS Grid ve animasyonlar.
-   **JavaScript (ES6+)**: SPA benzeri yönlendirme (router), Canvas API ve oyun mantığı.

## 🚀 Nasıl Çalıştırılır?

Bu projeyi yerel makinenizde çalıştırmak oldukça basittir:

1.  Bu repository'yi bilgisayarınıza indirin.
2.  `index.html` dosyasını herhangi bir modern tarayıcıda açın.
3.  Veya VS Code kullanıyorsanız **Live Server** eklentisi ile `Go Live` diyerek başlatın.

## 📂 Proje Yapısı

```text
MiniGames/
├── index.html          # Ana giriş noktası
├── style.css           # Global modern stiller
└── js/
    ├── main.js         # Oyun HUB yönetimi
    ├── router.js       # Sayfalar arası geçiş mantığı
    └── games/          # Her bir oyunun mantık dosyaları
        ├── snake.js
        ├── tictactoe.js
        ├── memory.js
        ├── flappy.js
        └── game2048.js
```
